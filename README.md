# Arquitectura PKI Empresarial

Este repositorio documenta cómo diseñar una PKI empresarial como **infraestructura estructural de confianza**, no como una simple implementación técnica.

El sitio fue ampliado para incluir una guía más completa de:

- Fundamentos arquitectónicos y modelo mental de diseño.
- Principios de aislamiento, segmentación y separación de roles.
- Modelos de referencia (2 niveles y jerárquico empresarial).
- Operación con controles, métricas y riesgos acumulativos.
- Hoja de ruta de madurez para evolución continua.

## Enfoque

La propuesta prioriza:

1. Claridad estructural sobre complejidad innecesaria.
2. Gobierno criptográfico explícito y auditable.
3. Resiliencia frente a compromiso de componentes críticos.
4. Diseño del ciclo de vida completo (emisión, renovación, revocación, contingencia).
5. Escalabilidad y adaptación a cambios regulatorios y criptográficos.
